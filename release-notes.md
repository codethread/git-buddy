### Installation

**Requires [`bun`](https://bun.sh/)**

```sh
curl -fsSL https://bun.sh/install | bash`
```

Download the `git-buddy.tar.gz` from the latest release, open it and then copy the git-buddy file somewhere in your `$PATH`; then run:

```sh
git-buddy --help
```
